# image-registration-reinforcement-learning
A synthetic image dataset generation and image-to-text model for recognising geometric shapes and their spatial relationships using deep learning.
This project focuses on generating a synthetic dataset of geometric shapes and training a model to describe images in natural language.


Key Features:
Controlled placement of objects (center, relative positions, overlap)




Tech Stack:
Python
PIL / OpenCV
NumPy / Pandas
TensorFlow / PyTorch
