# image-registration-reinforcement-learning
A synthetic image dataset generation and image-to-text model for recognising geometric shapes and their spatial relationships using deep learning.
This project focuses on generating a synthetic dataset of geometric shapes and training a model to describe images in natural language.
The dataset consists of images containing shapes such as circles, triangles, squares, rectangles, cylinders, and pyramids placed in different spatial positions (e.g., above, below, left, right, overlapping).
A neural network model is then trained to map these images to descriptive text outputs, enabling basic visual understanding and scene description.
Key Features
Automated dataset generation using Python (PIL/OpenCV)
Controlled placement of objects (center, relative positions, overlap)
CSV-based annotation for supervised learning
Image-to-text model using deep learning
Scalable pipeline for generating large datasets
Tech Stack
Python
PIL / OpenCV
NumPy / Pandas
TensorFlow / PyTorch
Use Case
This project demonstrates:
Fundamentals of computer vision
Basics of image captioning
Controlled dataset creation for ML models
